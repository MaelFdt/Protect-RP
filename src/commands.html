ACCUEIL COMMANDES POLITIQUE SUPPORT INVITER

Divertissement

/blague

Vous pouvez obtenir une blague francophone ou anglophone.

/avatar

Permet d'obtenir le lien de l'avatar d'un membre du serveur.

/chat

Très simple, cette commande retourne une image aléatoire d'un chat.

/météo

Obtenir les conditions météorologiques d'une ville.

/série

Cette commande vous permet d'obtenir les détails d'une série.

/film

Cette commande vous permet d'obtenir les détails d'un film.

Modération

/raid lock

Permet de vérouiller le chanel.

/raid unlock

Permet de dévérouiller le chanel.

/raid raidmode

Permet d'activer ou désactiver le mode anti raid.

/info

Permet d'obtenir toute les informations d'un joueur.

/purge

Supprimer jusqu'à 100 messages d'un channel d'un coup.

/warns avertir

Permet d'avertir un membre du discord.

/warns reset

Permet de supprimer tout les warns d'un joueur.

/warns liste

Permet d'obtenir la liste d'avertissement d'un joeuur.

/config2 lien

Permet d'activer ou désactiver la detection et supression des liens.

Fonctionnalités Roleplay

/config rpnames

Cette commande vous permet d'activer l'option de nom RP automatisé du bot dans un channel.

/config rpnamesrole

Si configuré, ce rôle sera attribué à tous les membres qui inscriront un message dans le canal de nom roleplay.

/config statuschannel

Si configuré, vous obtiendrez la liste de tout les joueurs connectés a votre serveur.

/config anonymous

Permet d'ajouter un salon dédier au messages anonyme.

/config anonymouslogs

Définir le salon textuel pour l'envoi des logs des salon anonymes.

/config douane douanechannel

Permet de définir un salon pour les douanes automatisé.

/config douane douanier

Permet de configurer le role autorisé a utiliser le systeme de whitelist.

/config douane immigrantrole

Permet de définir un role Immigrant pour les douanier.

/config douane whitelistrole

Permet de définir un role WhiteList pour les douanier.

/config rpip

Permet de définir l'adresse IP de votre serveur rp.

/config rpcheck

Permet de comparer les joueurs en ligne et ceux qui sont connectés a un vocal.

/rp joueur

Permet de rechercher un joueur en jeu par son nom, prénom ou son ID.

/rp serveurinfo

Permet d'obtenir les informations de serveur roleplay.

/giveaway créer

/roles

Permet de configurer le salon dans le quel seront traiter les avis staff.

/config2 suggestion

/config prefix

Permet de configurer le prefix que vous souhaiter utiliser sur le BOT.

/config welcomechannel

Permet de modifier le canal de bienvenue dans lequel un message automatisé sera envoyé lorsqu'un nouveau membre rejoint votre serveur.

/config welcomeimage

Permet de configurer l'envoi d'une image personnalisé lorsqu'un nouveau membre rejoint votre serveur Discord.

/config english

Utilisez la commande ci-dessus afin d'activer ou de désactiver l'option anglophone.

/config staffroleid

/config supportrole

Permet de configurer le rôle support.

/config logschannel

/config autochanmainid

Permet de configurer le salon à utiliser pour la création des canaux automatisés..

/config supportchannel

Permet de configurer le salon vocal par défault pour le support automatisés.

/config douane douanechannel

Permet de configurer configurer le salon de douane automatisés.

/config commandschannels

Permet aux membres d'utiliser les commandes personnalisés.

/commandes créer

Permet de créer une nouvelle commande.

/commandes modifier

Permet de modifier une commande.

/commandes supprimer

Permet de supprimer une commande.

/commandes désactiver

Permet de désactiver une commande.

/commandes raw

Permet d'obtenir le contenue brut d'une commande.

/commandes embed

Permet d'activer l'option embed sur une commandes personnalisé.

/commandes privée

Permet d'activer l'option privée sur une commandes.

/config ghostpings

Permet d'activer ou de désactiver la détection de ghost ping.

Sécurité

/giveaway créer

Permet de créer un giveaway.

/config captcha modifier

Permet de modifier la limite de tentatives avant d'etre expulser.

/config captcha créer

Permet de créer un captcha.

/config captcha captchalogs

Permet de configurer le channel dans le quel seront envoyés les logs liée au captcha.

Système de billets

/config ticketspanels

Permet de configurer créer un systeme de ticket.

/config ticketparent

Permet de configurer une nouvelle catégorie dans laquelle les nouveaux billets doivent être ouverts.

/transcript salon

Permet de transrire un ticket (via billet en cours).

/transcript numéro

Permet de transrire un ticket (via un numéro de billet).

/billets membre

Cette commande permet d'obtenir la liste des billets ouverts par un membre.

/ticket permissions rôle

Permet de modifier les permissions d'un rôle.

/ticket permissions admin

Permet de rendre le billet visible seulement aux admin et membres du billet.

/ticket permissions membre

Modifier les permission d'un membre.

/ticket nom

Permet de renommer le billet.

/ticket fermer

Permet de fermer le billet.
