Politique de confidentialité du bot Protect

Le bot Protect collecte plusieurs informations concernant les utilisateurs ainsi que leur contenu conformément aux politiques d'utilisation de l'api Discord.

Données collectées

Le bot Protect collecte plusieurs informations auprès de Discord.

Les données sont collectées lors d'interaction directe ou indirecte avec le bot ainsi que les divers types d'intéraction Discord. Ces données sont conservées de façon sécuritaire dans une base de données Firebase auprès de Google.

Concernant le site web Protec RP, aucune donnée n'est collectée.

Utilisation des données

Les données sont utilisées seulement pour les diverses fonctionnalités du bot Protect.

Nous employons des standard technique afin de protéger vos données contre toutes accès non autorisé aux données enregistrées.

Nous ne partageons aucune information fournie sans votre consentement.

Partage des données

Nous ne partageons aucune donnée avec des tiers parties. Cependant, certaines données collectées auxquels vous avez collaborés pourraient être vues par d'autres persones.

Destruction des données

En tout temps, vous pouvez faire parvenir une demande de destruction des données du serveur duquel vous êtes l'unique propriétaire ou de la destruction des données vous concernants.

Veuillez nous contacter afin de faire votre demande via l'une des options ci-dessous.

  • Par courriel
  • Via message privé sur Discord: Faster#0001

Sécurité

Nous utilisons des services externe sécuritaire afin d'augmenter la sécurité de vos données.

Mineurs

Le bot n'est pas conçu pour l'utilisation par des personnes de moins de 13 ans conformément aux politiques d'utilisation de Discord.

Changement concernant les politiques

Ces politiques peuvent être mises à jour à tout moment et ce, sans préavis.
